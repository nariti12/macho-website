# マチョクリッカー Cookie Clicker 仕様寄せ TODO

## 反映済み

- 建物の基本価格と毎秒生産量は、Cookie Clicker の主要建物カーブに近い値へ寄せる。
- 建物価格上昇率は Cookie Clicker と同じ `1.15` を維持する。
- Fractal Engine と Idleverse の間にある Javascript Console 相当として、`筋肉コンソール` を追加する。
- 仕上げ直しは `1 trillion` 相当から開始し、倍率計算は平方根ではなく三乗根カーブにする。
- ゴールデンプロテインの基本出現間隔は `5〜15分`、効果時間は `13秒`、Frenzy は `77秒 x7`、Click Frenzy は `13秒 x777` に寄せる。
- クリック時に毎秒生産量が乗る強化は強すぎたため、`1%` から `0.1%` に下げ、解放条件をダンベル所持数ベースへ遅らせる。

## 追加で確認・実装したいこと

- 建物アップグレードの価格倍率を、Cookie Clicker の各建物別アップグレード価格にさらに近づける。
- Cursor 系のクリック強化を、Cookie Clicker の Cursor / Mouse upgrade の細かい解放条件に寄せる。
- Golden Cookie 相当の効果抽選比率を、通常時・アップグレード取得後・季節イベント時で分岐させる。
- 実績数を増やし、Milk 相当の恒常ボーナスを入れるか検討する。
- Sugar Lump 相当の長期育成要素を入れるか検討する。
- 建物ごとのミニゲーム相当は、初期版では重くなるため未実装。必要なら別タスクで設計する。
- オフライン生産の上限や倍率を、Cookie Clicker のアップグレード取得状況に近づける。
- セーブ/インポート/エクスポートの表示を Cookie Clicker の設定画面に近づける。
- 統計画面に「総クリック数」「手動獲得」「建物別生産比率」「プレイ時間」をさらに追加する。
