# マチョクリッカー 終盤設計

最終検討: 2026-07-17

## オーバートレーニング危機

Grandmapocalypse 相当は `オーバートレーニング危機` とする候補です。終盤で自動発生させず、プレイヤーが研究を購入して開始する方式にします。

- 解放候補: マチョ田本人を1つ所有し、仕上げ直しを3回以上実行
- 通常時との差: ゴールデンイベントの抽選内容と背景ニュースが変化
- 終了手段: 一時停止と恒久終了を分け、誤操作で通常プレイへ戻れなくしない
- 実装判断: 終盤到達ユーザーの利用状況を確認してから実装する

## 疲労モンスター

Wrinkler 相当は `疲労モンスター` とし、生産の一部を一時保管する設計候補です。

- 1体あたり毎秒生産の一部を吸収
- クリックで追い払うと、吸収量へ小さな利息を付けて返却
- セーブ対象にし、再読込で消失させない
- スマホではキャラクタークリックの邪魔にならない外周へ配置

## 育成キャラクター

Krumblor 相当は新キャラクターを増やさず、既存の `マチョ猫` を育成枠として拡張します。実績数に応じた3段階アップグレードは実装済みです。追加する場合は、選択式の能力を同時に有効化しすぎず、付け替え可能にします。

## PWA判断

PWA化は現時点では保留します。ゲーム更新時に古いJavaScriptや画像がService Workerへ残るリスクがあり、まずモバイル継続率と再訪率を確認します。導入時はオフラインキャッシュ対象をゲーム静的アセットに限定し、ランキングAPIやサイト記事をキャッシュしません。
