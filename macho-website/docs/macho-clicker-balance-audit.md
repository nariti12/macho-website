# マチョクリッカー バランス監査

最終確認: 2026-07-17

## 設備カーブ

公式 Cookie Clicker の現行 `main.js` にある Cursor から You までを確認し、マチョクリッカーの20設備を同じ順番に対応させています。

- 基本価格: Cookie Clicker の各建物と同値
- 基本CPS: Cookie Clicker の各建物と同値
- 購入価格倍率: 全設備 `1.15`
- 対応確認: `npm run check:macho-clicker-balance`

この検証は設備定義をソースから直接読み、基準値との差があれば失敗します。アップグレード、ゴールデン効果、日課、季節イベントなどの追加倍率は別レイヤーとして扱います。

## 放置報酬

| 状態 | 1日離脱 | 1週間離脱 |
| --- | ---: | ---: |
| 通常 | 30分ぶん | 30分ぶん |
| 留守番トレーナー取得後 | 8時間30分ぶん | 8時間30分ぶん |

離脱期間そのものを全額付与しないため、1週間放置してもゲーム進行を飛ばしすぎません。

## 10分・30分・60分の進行比較

プレイヤー差を除くため、両ゲームへ同じクリック頻度とROI優先購入を適用した決定論的シミュレーションを実施しています。ゴールデンイベントや独自の日課ボーナスは含めず、設備カーブそのものを比較します。

| 経過 | Cookie Clicker CPS | マチョクリッカー CPS | Cookie設備数 | マチョ設備数 | 到達設備 |
| --- | ---: | ---: | ---: | ---: | --- |
| 10分 | 10.2 | 10.2 | 30 | 30 | 腹筋ローラー職人相当 |
| 30分 | 65.5 | 65.5 | 62 | 62 | バーベル部隊相当 |
| 60分 | 230.9 | 230.9 | 103 | 103 | プロテイン工房相当 |

設備の価格、CPS、購入倍率のいずれかが変わり、進行差が発生した場合は `npm run check:macho-clicker-balance` が失敗します。操作感は `npm run test:macho-clicker` でPC・スマホのクリック、購入、30分相当の進行、保存復元を確認します。

## 検証範囲

- 自動保証: 設備定義、購入倍率、10/30/60分の基礎進行、放置上限、PC/スマホ操作、保存復元、ゴールデン報酬、音声ファイル配信
- ランダム要素: 比較表から除外し、ブラウザテストでは乱数を固定して出現・獲得のみ確認
- 独自要素: 日課、季節、実績倍率、仕上げ直しはCookie Clickerとの同値比較対象ではなく、別レイヤーとして維持
