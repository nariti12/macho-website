# マチョクリッカー バランス監査

最終確認: 2026-07-17

## 設備カーブ

公式 Cookie Clicker の現行 `main.js` にある Cursor から You までを確認し、マチョクリッカーの20設備を同じ順番に対応させています。

- 基本価格: Cookie Clicker の各建物と同値
- 基本CPS: Cookie Clicker の各建物と同値
- 購入価格倍率: 全設備 `1.15`
- 対応確認: `npm run check:macho-clicker-balance`

この検証は設備定義をソースから直接読み、基準値との差があれば失敗します。アップグレード、ゴールデン効果、日課、季節イベントなどの追加倍率は別レイヤーとして扱います。

## 放置報酬

| 状態 | 1日離脱 | 1週間離脱 |
| --- | ---: | ---: |
| 通常 | 30分ぶん | 30分ぶん |
| 留守番トレーナー取得後 | 8時間30分ぶん | 8時間30分ぶん |

離脱期間そのものを全額付与しないため、1週間放置してもゲーム進行を飛ばしすぎません。

## 手動確認が必要な項目

数値の一致だけでは操作感を判定できないため、PCとスマホで10分・30分・60分の実プレイ比較は別途残します。ここでは「設備データが基準から意図せず変わっていないこと」と「放置報酬の上限」を自動保証します。
