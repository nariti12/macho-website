# マチョ田 キャラクターバイブル

最終更新: 2026-07-18

## 目的

マチョクリッカーの体型進化20段階を、別人の寄せ集めではなく、同じマチョ田が生活とトレーニングを改善していく連続した成長として制作する。

## 変えてはいけない特徴

- 日本人男性。正面を向いた全身立ち姿で、視線は正面。
- 横幅のある角張った顔、太く水平に近い黒い眉、小さめの黒い瞳、低く広い鼻、強い顎。
- 肌は健康的な中間色。段階によって肌色、人種、年齢を変えない。
- 髪は黒。基本形は短いトップと、耳の後ろから肩へ伸びる長い後ろ髪。
- 腕組みを基本姿勢とし、両足を肩幅程度に開く。手足の位置を毎回大きく変えない。
- 黒系のパンツと靴を基本にし、足元位置と人物の中心軸を全画像で揃える。
- 太い黒線、抑えたセル塗り、適度な陰影を使う。写実、3D、ドット絵へ途中で変えない。

## 段階的に変える特徴

- 0〜4: 乱れた生活から運動習慣ができるまで。猫背、腹部、髪、ひげ、服の汚れを段階的に改善する。
- 5〜9: 初心者から中級者へ。白い普段着から清潔なトレーニングウェアへ移り、肩・腕・胸の輪郭を少しずつ出す。
- 10〜14: 中級者から上級者へ。逆三角形、胸板、脚の厚みを増やし、姿勢と表情に自信を加える。
- 15〜19: 上級者から完成形へ。全身の厚み、絞り、ポージング、専用ウェア、控えめな演出を順番に加える。
- 一段階で変える主項目は1〜2個に絞り、隣接段階で急に別体型・別衣装へ飛ばさない。

## 制作仕様

- キャンバス: 720 x 1080 px。
- 形式: 透過PNG。背景、床、影の四角い面、文字、ロゴは含めない。
- 足元基準: 下端から24〜64pxの範囲。頭頂は上端から24〜72pxの範囲。
- 水平位置: 人物中心をキャンバス中央から左右5%以内に置く。
- 光源: 左上。顔、白い服、腕のハイライト方向を全段階で揃える。
- 外周: 白フリンジ、色付きハロー、切れた髪・靴を残さない。
- 被写体占有率: 透明部分を除く画素がキャンバスの18〜72%を目安とする。

| Stage | 名称 | この段階で変えるもの |
| ---: | --- | --- |
| 0 | 生活崩壊期 | ぼさぼさ髪、無精ひげ、猫背、汚れた大きめの服、メタボ体型 |

| 4 | 食生活改善 | 腹部を少し絞り、表情を明るくする |
| 5 | 脱メタボ開始 | 清潔なトレーニングウェアへ替える |
| 6 | 筋肉の芽 | 肩と上腕に最初の輪郭を出す |
| 7 | 初心者卒業 | 姿勢を直し、胸を張る |
| 8 | 細マッチョ入口 | 腹部と胸の輪郭を出す |
| 9 | 中級トレーニー | 肩幅と脚を一段階発達させる |
| 10 | 胸板覚醒 | 胸と上腕三頭筋を厚くする |
| 11 | 逆三角形 | 広背筋と肩のシルエットを強める |
| 12 | バルク期 | 全身の厚みを増やす |
| 13 | 仕上げ期 | 体脂肪を落とし、筋肉の境目を出す |
| 14 | 上級マッチョ | ウェアと表情を洗練する |
| 15 | ゴリマッチョ | 全身の大きさと重量感を強める |
| 16 | 大会準備 | ポージングと弱いリムライトを加える |
| 17 | 大会仕上がり | 絞りと血管表現を一段階強める |
| 18 | 伝説のマチョ田 | 専用ウェアと控えめなオーラを加える |
| 19 | 最終形態 | 現行完成形の顔と雰囲気を最高品質で統合する |

## 禁止事項

- 既存画像を拡大縮小・色替えしただけで別段階としない。
- 同じ画像を複数のStageへ割り当てない。
- 腕や指の本数、関節、服のボタン、左右の靴を破綻させない。
- 体型や生活状態を侮辱する文章・表情にしない。成長前後の落差は努力の達成感として表現する。
- 実画面、透過検査、隣接3段階の比較を通さず採用しない。

## 採用手順

1. 前段階、制作対象、最終段階の3枚を参照して生成する。
2. `npm run check:macho-evolution-assets`で透過、寸法、外周、占有率を検査する。
3. 前後3段階を並べ、顔・足元・光源・線の太さ・変化量を確認する。
4. PC、390pxスマホ、暗い背景、明るい演出で実画面確認する。
5. 合格後にだけ`bodyEvolutionStages`へ割り当てる。
